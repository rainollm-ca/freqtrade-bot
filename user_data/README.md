# Freqtrade User Data

This directory contains all Freqtrade configuration and data files.

- `config.json` - Main configuration file
- `strategies/` - Custom trading strategies
- `data/` - Downloaded market data for backtesting
- `logs/` - Freqtrade log files
- `notebooks/` - Jupyter notebooks for analysis
- `hyperopts/` - Hyperopt result files

**Credentials:** All API keys and secrets are stored in the config.json file (kept in private repo).
